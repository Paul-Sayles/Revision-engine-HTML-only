# Revision Machine (Prototype)

A lightweight, browser-based revision system designed for classroom and independent study use.

---

## Overview

Revision Machine is a simple question engine that allows students to:

- Select a topic
- Answer exam-style questions one at a time
- Receive immediate feedback
- Track their score at the end of a session

It is designed to feel similar to systems like Sparx, but with a much smaller scope and no dependency on external services.

---

## Purpose

- Support independent revision (in or out of lessons)
- Reinforce key knowledge through repeated practice
- Provide a reusable framework across multiple subjects
- Enable trust ownership (no subscriptions, no external platforms)

---

## Features (Version 1)

- Runs entirely in the browser (no installation required)
- Topic-based navigation
- One-question-at-a-time interface (slide-style)
- Multiple choice questions (MCQ)
- Immediate feedback (correct/incorrect + explanation)
- Automatic score summary at the end
- Randomised question order per session

---

## How to Use

1. Download or clone the repository
2. Open `index.html`
3. Double-click to launch in any web browser
4. Select a topic and begin

No internet connection or server required.
